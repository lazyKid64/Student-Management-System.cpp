This project is designed to perform the given actions to a students data according to the user's choice .
I have made it similar to my institutes criteria of managment and it can be changed according to the future needs .
currently i have added these actions which can be performed -->
1. View List of Students
2. Sort List by CGPA
3. Students Above CGPA
4. Count of Students Above CGPA
5. Average CGPA
6. Top N Students
7. Search by Student Name
8. Department Rankwise List
9. Grade Distribution
10. Probation List
11. Placement Eligibility
 
i have added a random data of 150 students currently in the students.txt file to depict the working of the system .
